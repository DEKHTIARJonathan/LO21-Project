Description détaillée

Gère l'ensemble des fonctionnalités d'export.

Cette classe implémentant le design pattern Singleton contient l'ensembles des stratégies d'exports. Elle contient aussi l'ensemble des factory d'export qui permettent d'appliquer la stratégie d'export à une note selon le type réel de cette note.
