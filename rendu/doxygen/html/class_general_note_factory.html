Description détaillée

Cette classe implémentant le design pattern Singleton contient l'ensembles des factory permettant de gérer les différents types de note. Elle permettra notamment, la création d'une nouvelle note d'un certain type, le chargement/sauvegarde d'une note et la destruction d'une note.
