Description détaillée

Classe abstraite représentant une note générique avec un id et un titre.

Cette classe abstraite hérite de QObject, ainsi, grâce au système de métaobjet de Qt, nous pourrons accéder dynamiquement au type de l'objet (sous la forme d'un QString). Cette fonctionnalité sera particulièrement pratique dans la fabrication/gestion des instances, dans les stratégies d'exports et dans la fabrication/gestion des vues.
